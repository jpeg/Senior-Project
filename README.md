Senior-Project
==============

Main repository for senior project code.

### Command line usage:
`./vasc [OPTIONS]`  
#### Optional arguments:
`-e` - Email address to send alerts to.  
`-u` - Username for Gmail account to send alerts from.  
`-p` - Password for Gmail account to send alerts from.  

### Required libraries to build:  
* libopencv-dev
